# ArchSetup

Script automating the steps taken to install and setup a normal ArchLinux machine with the following features:
- BlackArchLinux Repositories
    - GoBuster
    - Dirb
    - Airoscript
    - John 
    - Hashcat 
    - SQLmap 
    - Hydra
- NMap
- Terminator
- Snapcraft:
    - Spotify
    - Notion
    - Telegram
    - Discord
    - VSCode
    - Postman
    - Slack
    - VLC
- Python
- PIP
